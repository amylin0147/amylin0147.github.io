About Scrabble

Scrabble was a java project demonstrating object-oriented analysis, object-oriented design, and GUI implementation for the class 15-214. The rules follow that in the wikipedia page with the addition of "special tiles". Special Tiles are used to disturb the game play and create unexpected changes to the game's state.
