About Scrabble

Scrabble was a java project demonstrating object-oriented analysis, object-oriented design, and GUI implementation for the class 15-214. The rules follow that in the wikipedia page with the addition of "special tiles". Special Tiles are bought with your points and used to disturb the game play, creating unexpected changes to the game's state.

The design process included creating a UML domain model, a system sequence diagram, interaction diagrams, and an object model. Design goals included extensibility, information hiding, and robustness. Testing was done with JUnit.
