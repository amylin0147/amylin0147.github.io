About Map Reduce

Map Reduce was a 15214 assignment to implement a distributed map/reduce framework. Map/reduce is a programming model for processing large data sets, typically implemented by distributing a computation across multiple worker servers where the data is stored.

In this project, I practiced network programming with Sockets, practiced parallel and concurrent programming in Java, and learned about the challenges of building a distributed system.
